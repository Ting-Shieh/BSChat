## 1. 模組目的（一句話複習）

企業版 Admin **發布並管理**「薄商務身份 stub」（姓名、公司、職稱、職責關鍵字、**外部名片 URL**），寫入 **Pool B（public_directory）**，供 **Pro+ 使用者跨池搜尋**（M5b）；**不做**完整電子名片宿主。
**對應 PRD**：§11.3 E、§11.5 Stage 2、§13.1~13.2（背景）、DDR-58~62、DDR-75~80（正向追溯 ✅）。

> **敘事權威**：**§11 v2.4（Free｜Pro｜企業版）**。Pro = 讀 Pool B；企業版 = 寫 Pool B；跨池搜尋為 Pro 內建能力（DDR-76、DDR-K11-02）。
