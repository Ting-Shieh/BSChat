## 1. M2 使用者流程

### 1.1 Onboarding + 首次收錄（Aha Moment First）

```mermaid
flowchart TD
    A[註冊完成] --> B[收錄 Tab：空狀態]
    B --> C{選擇收錄方式}
    C -->|連拍| D[相機連拍模式]
    C -->|貼連結 P1| E[貼上 URL]
    C -->|掃 QR P1| F[QR Scanner]
    D --> G[背景 Upload + OCR]
    E --> G
    F --> G
    G --> H{≥3 張 OCR 完成?}
    H -->|是| I[Modal：現在試試搜尋]
    H -->|否| J[繼續收錄提示]
    I --> K[搜尋 Tab]
    J --> D
```

### 1.2 展覽連拍主流程（P0）

```mermaid
flowchart TD
    A[點擊收錄 Tab] --> B[相機全屏]
    B --> C[連續拍照]
    C --> D[每張：底部缩略图 + upload 进度]
    D --> E{Upload 失败?}
    E -->|是| F[该张 retry 标记]
    E -->|否| G[背景 OCR]
    F --> C
    G --> H[卡片列表显示状态]
    H --> I{用户点结束?}
    I -->|是| J[关闭 Session]
    I -->|30min idle| J
    J --> K[Session 摘要页]
```

### 1.3 延後確認流程（P0）

```mermaid
flowchart TD
    A[待確認 Tab badge] --> B[待確認列表]
    B --> C[卡片：原图 + 姓名/公司/抬頭]
    C --> D{用户操作}
    D -->|编辑 + 确认| E[CONFIRMED → M3]
    D -->|右滑确认| E
    D -->|左滑跳过| F[保持 PENDING 仍可搜尋]
    D -->|稍後| B
    E --> G[Toast：已確認 ✓]
```

### 1.4 電子名片匯入（P1）

```mermaid
flowchart TD
    A[收錄 Tab] --> B{入口}
    B -->|貼連結| C[URL 输入框]
    B -->|掃 QR| D[QR 全屏]
    C --> E[解析中 spinner]
    D --> E
    E --> F{成功?}
    F -->|是| G[成功页 + 已收進名片庫]
    F -->|否| H[错误页 + 引导手动新增]
    G --> I[查看 / 继续收录]
```

### 1.5 錯誤恢復流程

```mermaid
flowchart TD
    A[错误发生] --> B{类型}
    B -->|Upload retry| C[Top banner：重试中 2/3]
    B -->|OCR failed| D[卡片标 OCR_FAILED + 原图]
    B -->|Duplicate| E[Toast：可能扫过 + 仍要收录]
    B -->|Import failed| F[Modal：格式不支持]
    D --> G[待確認：手动填 3 栏]
    E --> H{仍要收录?}
    H -->|是| I[Force 新 card]
    F --> J[手动新增 / 换连结]
```

### 2.4 待確認列表（A2：折疊「稍後再核對」）

**Badge** = 「需核對」數量（`pending_review` 且未跳過）；不含「稍後再核對」與「已自動確認 · 可選核對」。

```
┌─────────────────────────────────────┐
│  需核對 · 2                          │
│  請確認姓名、公司、抬頭三欄           │
├─────────────────────────────────────┤
│ ┌─────────────────────────────────┐ │
│ │ [原图]   王小明 · ABC Tech  🟠   │ │
│ │                    [稍後] [刪除] │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│  ▼ 稍後再核對 · 3（預設折疊）         │
│  （仍 pending_review，仍可搜尋）      │
├─────────────────────────────────────┤
│  🔍    📇    [➕]    ✓②   👤        │  badge=需核對數
└─────────────────────────────────────┘
```

**不在此 Tab 顯示**：`auto_accepted` / `confirmed` 直接進名片庫；確認後從列表移除。

**欄位規則**（DDR-18）：
- **僅 3 欄可編輯**：姓名、公司、抬頭
- 電話/Email 詳情頁 readonly grey
- **稍後再確認**（跳過）→ 移入折疊區；狀態仍 `pending_review`；**仍可搜尋**（R-12）
- **刪除** → raw card + contact 一併 soft-delete；需二次確認
- P1：右滑確認 · 左滑跳過（手勢）

### 2.5 待確認 — 單張詳情

```
┌─────────────────────────────────────┐
│  ←  返回列表                         │
├─────────────────────────────────────┤
│      [名片原图 full width]           │
│  姓名   [________________] 🟠       │
│  公司   [________________]           │
│  抬頭   [________________] 🟠       │
│  ── 以下 AI 已處理 ──                │
│  電話 / Email（readonly grey）       │
│  ┌─────────────────────────────┐   │
│  │        確認 ✓                │   │  Primary
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │      稍後再確認              │   │  Secondary（僅 pending）
│  └─────────────────────────────┘   │
│         刪除這張名片                 │  destructive link
└─────────────────────────────────────┘
```

**刪除確認對話框**：
- 標題：刪除這張名片？
- 內文：將從名片庫與待確認移除，之後搜尋不到這位聯絡人。
- 顯示姓名 · 公司摘要

**Toast**：
- 確認：已確認 ✓
- 跳過：已跳過，仍可在搜尋中找到這位聯絡人
- 刪除：已刪除

## 3. 錯誤狀態設計（SA/SD L4 #4 對照）

| 錯誤碼 | UI 模式 | 文案 | 恢復動作 |
|--------|---------|------|---------|
| UPLOAD_RETRYING | Bottom banner | 上传中，重试 (2/3)… | 自動 |
| UPLOAD_FAILED | 缩略图 ❌ + tap | 上传失败 | [重试] [删除] |
| OCR_FAILED | 卡片 badge 红 | 自动辨识失败 | → 待確認填 3 栏 |
| OCR_QUEUED_DELAYED | 卡片 badge 灰 | 处理中，预计 12 分钟 | 無需操作 |
| DUPLICATE_WARNING | Toast 5s | 3 天前可能扫过这张 | [仍要收录] |
| UNSUPPORTED_FORMAT | Modal | 此连结格式不支持 | [手动新增] [换连结] |
| IMPORT_FAILED | Inline error | 无法解析此 QR | [改贴连结] |
| SESSION_ZERO_SUCCESS | 摘要页 empty | 本次 0 张成功 | [检查光线重试] |
| VERSION_CONFLICT 409 | Toast | 资料已更新，请刷新 | 自動 refresh |
